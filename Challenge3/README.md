# Simple React Website

A basic website built with HTML, CSS, and JavaScript using React via CDN.

## Files

- `index.html` - Main HTML file with React CDN links
- `style.css` - Styling with modern design
- `app.js` - React components with hooks

## Features

- **Counter** - Increment/decrement/reset using useState
- **Todo List** - Add, complete, and delete tasks
- **Live Clock** - Real-time clock using useEffect

## How to Run

Just open `index.html` in your browser. No build step needed!
